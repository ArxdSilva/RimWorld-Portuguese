# TRADUÇÃO RIMWORLD Português #
#################################

Versão em português para o RimWorld Alpha 3.
Participe do projeto em: https://github.com/Ribas-7/RimWorld-pt

===INSTALAÇÃO===
1- Dentro do arquivo encontra-se a pasta "Mods" (/Mods/Core/Languages/Portuguese).
2- Cole a pasta Mods onde o seu RimWorld foi instalado, no caso; C:\RimWorld Alpha 3, na mesma onde se encontra o arquivo.exe "Rimworld413win".
3- Substítua todos os arquivos e pronto.
4- vá para a tela Options no jogo, e mude o idioma (language) para Portuguese.
